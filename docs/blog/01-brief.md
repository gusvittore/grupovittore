# Blog Grupo Vittore — Brief

## Papel no site

O blog fica dentro do domínio principal, na rota `/blog`. Ele funciona como um centro de conteúdo estratégico do Grupo Vittore, apoiando autoridade, educação empresarial e navegação para as frentes da empresa.

A página inicial deve transmitir clareza, organização, inteligência empresarial e sofisticação institucional. O blog não deve parecer portal de notícias, vitrine de ofertas ou landing page agressiva.

## Objetivos

- explicar temas de vendas, marketing, materiais gráficos, tecnologia e crescimento empresarial;
- ajudar gestores e empresários a reconhecer gargalos e oportunidades;
- fortalecer a autoridade institucional do Grupo Vittore;
- criar links internos para `/assessoria-comercial` e `/materiais-impressos`;
- preparar uma base local simples para futuros artigos em `/blog/[slug]`.

## Direção editorial

O conteúdo deve ser claro, didático, estratégico e profissional. A escrita deve apoiar decisões melhores sem prometer resultados garantidos, usar urgência artificial ou transformar cada artigo em uma oferta comercial.

## Arquitetura

- `/blog` — página inicial do blog;
- `/blog/[slug]` — estrutura futura para artigos individuais;
- `/blog/categoria/[categoria]` — estrutura futura para categorias.

Não usar subdomínio e não criar CMS, banco de dados ou API nesta etapa.
